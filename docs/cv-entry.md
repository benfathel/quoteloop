# CV Entry

## Short Version

**QuoteLoop - Full-Stack SaaS Platform**  
Built a mobile-first quote follow-up and booking platform for trade contractors using Next.js, TypeScript, Prisma, PostgreSQL, Stripe, NextAuth, and n8n. Implemented quote tracking, automated follow-up workflows, subscription gating, customer intake forms, availability scheduling, booking management, and notification flows.

## Bullet Version

- Built QuoteLoop, a full-stack SaaS app that helps trade contractors track quotes, automate customer follow-ups, and convert accepted quotes into bookings.
- Implemented a Next.js 14 App Router product with TypeScript, Tailwind CSS, Prisma, PostgreSQL, NextAuth phone OTP login, Stripe subscriptions, and n8n workflow automation.
- Designed core business flows including quote status tracking, custom follow-up timing, quote expiration, customer intake forms, availability slots, booking confirmation, rescheduling, cancellation, and in-app notifications.
- Added Free/Plus plan logic, usage limits, billing portal integration, protected dashboard routes, public booking links, and cron jobs for operational automation.

## LinkedIn / Portfolio Version

QuoteLoop is a private full-stack SaaS project for solo trade contractors. It solves a common workflow problem: contractors send quotes, forget to follow up, and lose jobs. The app lets contractors manage quotes, automate follow-up workflows, collect customer booking details, and handle appointments from a mobile-first dashboard.

Built with Next.js, TypeScript, Prisma, PostgreSQL, Stripe, NextAuth, Tailwind CSS, and n8n.
